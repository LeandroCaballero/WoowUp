
# WoowUp - Challenge

### Consideraciones
- El programa se maneja por medio de la consola
- Las alertas tienen una expiración de 2 min
- El manejo de errores no está implementado
- El ejercicio de SQL se encuentra en la carpeta "sql"

## Instalación

```bash
 cd challenge
 npm install
 npm run dev
```
    
## Test

```bash
  npm run test
```
